Live site: [devagurunanduri.netlify.app](https://devagurunanduri.netlify.app)

## About

This portfolio showcases projects, experience, skills, certifications, and achievements in the fields of Artificial Intelligence, Machine Learning, Computer Vision, and IoT. It is designed to be fast, fully responsive, and deployable with zero configuration.

## Tools and Technologies

**Languages**
- HTML5
- CSS3
- JavaScript (vanilla, no frameworks)

**Fonts**
- Bebas Neue — display headings
- DM Sans — body text
- JetBrains Mono — labels, tags, monospace elements
- All loaded via Google Fonts

**Form Handling**
- Formspree — contact form submissions delivered to email inbox, no backend required

**Animations and Interactions**
- CSS keyframe animations — hero entrance, orb floats, scroll indicator
- IntersectionObserver API — scroll reveal for all sections
- CSS transitions — hover effects on cards, buttons, links
- JavaScript mouse tracking — radial glow effect on project cards
- Custom CSS cursor with lag ring (desktop only, pointer: fine media query)

**Layout**
- CSS Grid — projects, skills, about, extras, achievements sections
- CSS Flexbox — nav, buttons, tags, contact links
- CSS custom properties (variables) — consistent theming throughout
- Fully responsive with breakpoints at 960px and 500px
- Mobile hamburger menu with animated open/close state

**Hosting**
- Netlify — deployed via manual drag and drop, no CLI required

**Resume**
- Hosted on GitHub raw file URL with HTML download attribute for direct PDF download

## How to Run Locally

Download or clone the repository, then open `index.html` directly in any browser. No install, no build step, no dependencies.

```
open index.html
```

---

## How to Deploy on Netlify

1. Go to netlify.com and sign in
2. Click Add new site, then Deploy manually
3. Drag and drop the index.html file onto the upload area
4. The site is live instantly at a generated netlify.app URL
5. To customize the URL, go to Site settings, Site details, Change site name
